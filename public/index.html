Timestamp microservice

Given a valid date endpoint returns an object containing date in both unix and natural formats else returns an object with null values

eg:

Valid endpoints

  • /December 1, 2014 or /December%201,%202014
  • /1417372200

    • Returns

    {'unix':1417372200,'natural':"December 01, 2014"}

Invalid endpoint

  • /hello

    • Returns

    {'unix':null,'natural':null}
